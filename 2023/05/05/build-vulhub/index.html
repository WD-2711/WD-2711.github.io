vulhub搭建

 最近护网面试，面试官建议可以打一下靶场，所以就打算搭一个vulhub，供我和翔哥打。

 教程中，docker-compose up -d 是一个命令，用于在 Docker Compose 环境中启动容器。这个命令会根据 docker-compose.yml 文件中的配置启动容器，并且在后台运行（-d 参数表示 “detached mode”，即后台模式）。具体来说，docker-compose.yml 文件是一个声明式的配置文件，用于定义和配置多个 Docker 容器的运行方式。在这个文件中，可以指定容器的镜像、端口、环境变量、挂载的卷等等。docker-compose up -d 命令会读取这个文件，按照其中的配置启动容器，并将其放在后台运行。如果一切顺利，docker-compose up -d 命令会返回一些容器的 ID，表示容器已经在后台启动。可以使用 docker ps 命令查看这些容器的运行状态。

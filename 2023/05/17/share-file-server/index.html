使用python建立HTTP文件服务器

很多时候我们需要快速地查看服务器上的文件，或者将文件分享给别人。
在Linux服务器上，这个需求用一行指令即可实现，首先cd到需要分享的文件夹，然后执行：

  • Python 2: python -m SimpleHTTPServer 8080
  • Python 3: python -m http.server 8080
    这里指定的端口是8080，在浏览器中使用 http://yoursite:8080 即可访问。
